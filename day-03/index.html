Q1:What are promises and why do we need them?

ANS: Promise are used to handle asynchronous operations in javascript.Promise is easy to handle the multiple asynchronous operations.Mutiple callback function would create callback hell that leads to unmanageable code. So, promise provide better error handling than callback.

                            })
                            
                            function funcPromise(){
                                          return new Promise((resolve,reject)=>{

Q2: What is promise Chaining?

Ans:It is technique in javascript to execute multiple asynchronous operations in a specific order by chaining promise together using then method. 

                function display(num,timeout){
                      return new Promise((res,rej)=>{
                            setTimeout(() => {
                                  console.log(num);
                                  res("Promise resolved!!")
                            }, timeout);
                      })
                }
                
                display(1,1000)
                .then(()=>display(2,2000))
                .then(()=>display(3,3000))
                .then(()=>display(4,4000))
                .then(()=>display(5,5000))
                .then((data)=>console.log("Promise completed.."))

Q5:How many operators do we have in JS?

ANS: There are 7 operators in Js.


1. Arithmetic operator -Addition,subraction,multiply,division,exponential,modulus are have.

2. Assignment operators- =,+=,-=,*=,/=,%= are have.

3.Bitwise operators - Bitwise OR, bitwise AND ,bitwise NOT, left shift(<<) and right shift(>>)are have.

4. Comparison operator - Equal(==) , strict equal(===), not equal(!=) , strict not equal(!==) , greater than,less than,greater than equal,less than equal are have..

5. Logical operator - AND(&&) , OR(||) , NOT (!) are have.

6. Ternary operator - (Condition)?"Execute true" : "Execute false" . THis is the syntax of this operator.

7. TypeOf operator - it return datatype.
